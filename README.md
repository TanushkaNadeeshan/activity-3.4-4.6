# TV Energy Consumption Data Story

## Data Story
This project presents a data story about TV energy consumption. The audience is people who want to buy a television and understand how screen technology and screen size affect annual energy use.

### Audience
The target audience is TV buyers who want to compare different screen technologies and make better purchasing decisions.

### What they want to know
They want to know whether some TV technologies use more energy than others, and whether larger TVs consume more electricity.

### Key findings
- LCD and LCD (LED) TVs are more common in the dataset.
- OLED TVs are larger on average in this dataset.
- Energy consumption increases as screen size increases.
- This suggests screen size is an important factor when comparing energy use across technologies.

## About the Data

### Data Source
The dataset contains information about televisions, including screen technology, screen size, and annual energy consumption.

### Data Processing
The data was explored and processed using KNIME and spreadsheet tools to group categories, calculate averages, and create visualisations.

### Privacy
The dataset contains product information only, so there are minimal privacy concerns.

### Accuracy and Limitations
The dataset may not represent the full TV market. Some categories may have fewer examples, and not every factor affecting energy use is included.

### Ethics
The data was presented carefully to avoid misleading comparisons. In particular, screen size was considered when comparing screen technologies.
